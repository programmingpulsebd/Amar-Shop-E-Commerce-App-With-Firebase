
#🛒 E-Commerce App (Flutter + Firebase)

🛒 E-Commerce App (Flutter + Firebase)

📌 🔐 Authentication Module

User authentication related সব ফিচার:

✅ Signup (Email / Phone)
✅ Login
✅ Forgot Password (Email Reset)
✅ Change Password
✅ Logout
👤 👤 User Profile Module

User এর সব ইনফো manage করার জন্য:

👤 View Profile
✏️ Edit Profile (Name, Phone, Address, Image)
🖼 Profile Picture Upload
🔐 Change Password
🛍 Product Module

Product related main system:

📦 Product List (Grid/List View)
🔍 Search Product
📄 Product Details Page
⭐ Product Rating & Review
🏷 Discount Price / Offer Show
📂 Category Module

Product filtering system:

📁 Category List
📦 Category-wise Product Show
🔎 Sub-category Support (Optional)
❤️ Wishlist (Favorites)

User favorite product save:

❤️ Add to Favorite
❌ Remove from Favorite
📌 Wishlist Page
🛒 Cart Module

Shopping cart system:

➕ Add to Cart
➖ Update Quantity
❌ Remove Item
💰 Total Price Calculation
📦 Order Module

Order system (important part):

📝 Place Order
📜 Order History
📦 Order Details
🚚 Order Status (Pending, Shipped, Delivered)
❌ Cancel Order
💳 Payment Module (Optional)
💵 Cash on Delivery (COD)
💳 Online Payment (Stripe / SSLCommerz / bKash)
🔔 Notification Module
🔔 Order Update Notification
🔥 Offer / Discount Notification
📩 Push Notification (Firebase)
